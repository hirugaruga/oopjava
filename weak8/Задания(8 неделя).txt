Тема: Разработка классов. Цель: научиться разрабатывать и 
реализовывать классы, согласно стандартам языка Java
Внимание!!! речь идет именно о структуре КЛАССА решающего упражнение
НЕ системы классов (класс пускатель не считается) БЕЗ иерархии наследования.
Указания:
    Использовать перечисления (где это оправдано). 
    Рассматривать возможность декомпозиции разрабатываемых 
классов - на классы - данные (реализуют стратегию хранения
и вычисления производных характеристик присущих объекту) и 
классы - действия - реализуют операции над объектами.
    Создать классы, спецификации которых приведены ниже. Определить конструкторы
и методы setТип(), getТип(), toString().  
    В каждом классе, обладающем информацией,
должно быть объявлено несколько конструкторов. Перечисление массив/список/множество 
- означает, что для реализации коллекции объектов вы можете использовать
либо стандартные массивы, либо библиотечные контейнеры (списки, множества , вектора)
в последнем случае вам придется САМОСТОЯТЕЛЬНО разобраться с особенностями их использования.
    Реализовать методы сложения, вычитания, умножения и деления объектов (для
тех классов, объекты которых могут поддерживать арифметические действия):

1. Абитуриент: id, Фамилия, Имя, Отчество, Адрес, Телефон, Оценки.
Создать массив объектов. Вывести:
a) список абитуриентов, имеющих неудовлетворительные оценки;
b) список абитуриентов, у которых сумма баллов выше заданной;
c) выбрать заданное число n абитуриентов, имеющих самую высокую
сумму баллов (вывести также полный список абитуриентов, имеющих
полупроходную сумму).

2. Автобус: Фамилия и инициалы водителя, Номер автобуса, Номер маршрута,
Марка, Год начала эксплуатации, Пробег.
Создать массив объектов. Вывести:
a) список автобусов для заданного номера маршрута;
b) список автобусов, которые эксплуатируются больше заданного срока;
c) список автобусов, пробег у которых больше заданного расстояния.

3. Квадратное уравнение. Реализовать методы для поиска
корней, экстремумов, а также интервалов убывания/возрастания. Создать
массив/список/множество объектов и определить 
наибольшие и наименьшие по значению корни.

4. Треугольник на плоскости. Определить площадь и периметр
треугольника. Создать массив/список/множество объектов и подсчитать 
количество треугольников разного типа (равносторонний, равнобедренный, 
прямоугольный, произвольный). Определить для каждой группы
наибольший и наименьший по площади (периметру) объект.

5. Определить класс Комплекс, действительная и мнимая часть которой
представлены в виде Рациональной Дроби. Создать массив/список/множество
размерности n из комплексных координат. Передать его в метод,
который выполнит сложение/умножение его элементов.
